LiquidJS

Liquid template engine for Node.js and browsers.

View on GitHub for Production for Development

Features

  • Support both Node.js and browsers. Here's a demo: https://jsfiddle.net/6u40xbzs/
  • Fully compatible to shopify, with all tags and filters implemented
  • In pure JavaScript with any-promise as the only one dependency

Installation

  • Node.js 
    npm install --save liquidjs
  • Browser (production) 
    <script src="//unpkg.com/liquidjs/dist/liquid.min.js"></script>
  • Browser (development) 
    <script src="//unpkg.com/liquidjs/dist/liquid.js"></script>

    • Node.js Demo

    var Liquid = require('liquidjs');
var engine = Liquid();

engine
    .parseAndRender('{{name | capitalize}}', {name: 'alice'})
    .then(console.log);

// outputs 'Alice'
